# Environment Variables Setup

Copy the content below to create your `.env.local` file:

```bash
# ============================================
# AI Visual Content Reviewer - Environment Variables
# ============================================
# Copy this content to .env.local
# cp .env.example .env.local
# OR create .env.local manually and paste this content

# ============================================
# VISION MODEL SELECTION
# ============================================
# Choose which AI model to use for image analysis
# Options: 'claude', 'openai', 'gemini', or 'auto' (default)
# 
# - 'claude': Use Claude 3.5 Sonnet (best at following instructions, fewer hallucinations)
# - 'openai': Use GPT-4o (best OCR accuracy, precise analysis)
# - 'gemini': Use Google Gemini (free tier with generous limits)
# - 'auto': Automatically tries models in order: Claude → OpenAI → Gemini → Hugging Face → Groq
#
# Default: auto
VISION_MODEL=auto

# ============================================
# CLAUDE 3.5 SONNET (Anthropic)
# ============================================
# Recommended for: Best accuracy, fewer hallucinations, excellent at following complex instructions
# Free tier: $5 in credits to start (enough for testing)
# Get your API key at: https://console.anthropic.com/
# 
# Steps to get API key:
# 1. Go to https://console.anthropic.com/
# 2. Sign up or log in
# 3. Navigate to API Keys section
# 4. Create a new API key
# 5. Copy the key (starts with 'sk-ant-')
ANTHROPIC_API_KEY=your_anthropic_api_key_here

# ============================================
# GPT-4o (OpenAI)
# ============================================
# Recommended for: Best OCR (text reading), precise image analysis
# Free tier: $5 in credits to start (enough for testing)
# Get your API key at: https://platform.openai.com/api-keys
#
# Steps to get API key:
# 1. Go to https://platform.openai.com/api-keys
# 2. Sign up or log in
# 3. Click "Create new secret key"
# 4. Copy the key (starts with 'sk-')
# 5. Important: Save it immediately, you won't be able to see it again!
OPENAI_API_KEY=your_openai_api_key_here

# ============================================
# GOOGLE GEMINI
# ============================================
# Recommended for: Free tier with generous limits, good fallback option
# Free tier: Generous limits available
# Get your API key at: https://aistudio.google.com/app/apikey
#
# Steps to get API key:
# 1. Go to https://aistudio.google.com/app/apikey
# 2. Sign in with your Google account
# 3. Click "Create API Key"
# 4. Select or create a Google Cloud project
# 5. Copy the API key
GEMINI_API_KEY=your_gemini_api_key_here

# ============================================
# HUGGING FACE (Optional - Fallback)
# ============================================
# Used as fallback when primary vision models fail
# Free tier: Available with rate limits
# Get your token at: https://huggingface.co/settings/tokens
#
# Steps to get token:
# 1. Go to https://huggingface.co/settings/tokens
# 2. Sign up or log in
# 3. Click "New token"
# 4. Token Type: Select "Read" token (simplest option)
#    OR "Fine-grained" token with "Make calls to Inference Providers" permission
# 5. Copy the token (starts with 'hf_')
HF_API_TOKEN=your_huggingface_token_here

# ============================================
# GROQ (Optional - Last Fallback)
# ============================================
# Used as last resort when all vision APIs fail (text-only, less accurate)
# Free tier: Available with rate limits
# Get your API key at: https://console.groq.com/keys
#
# Steps to get API key:
# 1. Go to https://console.groq.com/keys
# 2. Sign up or log in
# 3. Click "Create API Key"
# 4. Copy the API key
GROQ_API_KEY=your_groq_api_key_here

# ============================================
# NOTES
# ============================================
# - You need at least ONE API key from the vision models above (Claude, OpenAI, or Gemini)
# - Hugging Face and Groq are optional fallbacks
# - All models have free tiers to start testing
# - Claude and OpenAI give $5 in credits initially
# - Gemini has generous free tier limits
# - Set VISION_MODEL to test different models and compare results
# - The 'auto' mode will automatically use the best available model
```

## Quick Start

1. Create `.env.local` file in the project root:
   ```bash
   touch .env.local
   ```

2. Copy the content above into `.env.local`

3. Get at least one API key:
   - **Claude**: https://console.anthropic.com/ ($5 free credits)
   - **OpenAI**: https://platform.openai.com/api-keys ($5 free credits)
   - **Gemini**: https://aistudio.google.com/app/apikey (free tier)

4. Replace `your_*_api_key_here` with your actual API keys

5. Set `VISION_MODEL` to test different models:
   - `VISION_MODEL=claude` - Best accuracy
   - `VISION_MODEL=openai` - Best OCR
   - `VISION_MODEL=gemini` - Free tier
   - `VISION_MODEL=auto` - Auto-select (default)

6. Restart your dev server:
   ```bash
   npm run dev
   ```
